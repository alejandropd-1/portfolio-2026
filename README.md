## 🚀 Tech Stack

- **Core**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Logic**: [React 18](https://reactjs.org/)
- **Styling**: SASS / SCSS Modules (Methodology: **BEM**)
- **Content**: MDX (Markdown for the "Compiled Soul" editorial style)
- **Animations**: [Motion for React](https://motion.dev/) (Framer Motion)
- **Icons**: [Lucide React](https://lucide.dev/)

---

## 📂 Project Structure

```bash
src/
├── app/            # Next.js App Router (Routes & Page Logic)
├── components/     # Reusable UI Components (SyntaxCard, Tag, Navigation, etc.)
├── content/        # MDX Content Files (Projects, Jobs, MDX Definitions)
├── helpers/        # Utility functions and formatting
├── hooks/          # Custom React hooks
└── styles/         # Global & Modular Design System
    ├── abstracts/  # Design Tokens (Colors, Sizes, Mixins)
    ├── base/       # Global Styles & Standard Typography
    ├── components/ # Styles for modular UI components
    └── pages/      # View-specific styles (Home, Project Detail, etc.)
```

## 🎨 Aesthetic Refactor (2026 Calibration)

The project has undergone a rigorous aesthetic calibration to achieve a "Pixel-Perfect" editorial look:

- **Standardized Typography**: Unified header scales across all views (5xl Mobile / 8xl Desktop) using *Jost* and *Roboto Mono*.
- **Glassmorphism System**: High-fidelity `@include glass` mixins with fine-tuned backdrop-filters and border-lights.
- **Terminal Simulations**: macOS-inspired window controls for MDX code blocks, technical specs, and project archives.
- **Color Calibration**: Strictly enforced color palette (`--clr-brand-*`) across cards, sidebars, and UI tokens.
- **Editorial Prosa**: Custom MDX styling for "The Challenge" and "The Solution" sections, including seamless vertical-bordered boxes.

## 🛠️ Development

### Prerequisites
- Node.js (Latest LTS)
- npm or pnpm

### Installation
```bash
# Install dependencies
npm install

# Set up environment variables
# Copy .env.example to .env.local and add your GEMINI_API_KEY
cp .env.example .env.local

# Run development server
npm run dev
```
